Финатлон

Веб-платформа для финансовых олимпиад

Главная О проекте Участники Журнал Ресурсы

Описание

Проект призван решить проблему, которая заключается в разрозненности информации одного проекта - Финатлон. Имеются три разных сайта, которые принадлежат данному проекту, однако куда удобнее было бы сделать единую веб-платформу со всеми функциями, содержащимися отдельно на каждом из сайтов: fin-olimp.ru, fin-victory.ru, finatlonforum.ru. При этом требуется решить проблемы слабого уровня вовлеченности аудитории

Цель проекта

Совершенствовать и продвигать существующие проекты по финансовой грамотности, создавая единую платформу для участников, партнеров и спонсоров, а также развивать Telegram-канал, VK-сообщество для постоянной коммуникации
