# СВЕДЕНИЯ О ПРОЕКТЕ Финатлон

## Цель проекта — совершенствовать и продвигать существующие проекты по финансовой грамотности, создавая единую платформу для участников, партнеров и спонсоров, а также развивать Telegram-канал и ВК-канал для постоянной коммуникации.
Проект направлен на:
объединение разрозненных проектов в единую веб-платформу для удобства участников;
автоматизацию сбора и обработки заявок на участие в олимпиадах с возможностью загрузки документов;
развитие регулярной коммуникации с аудиторией через Telegram-канал и сообщество ВКонтакте;
создание базы потенциальных партнеров и спонсоров для дальнейшего привлечения ресурсов;
повышение доступности и привлекательности олимпиад по финансовой грамотности для школьников и студентов.

## Основные задачи команды проекта

Развитие у обучающихся навыков командной работы;
Формирование навыков проектной работы;
Повышение у обучающихся мотивации к самообразованию;
Обеспечение освоения обучающимися основных норм профессиональной деятельности;
Получение обучающимися опыта использования основных профессиональных инструментов при решении нестандартных задач в рамках проектов;
Развитие навыков публичных выступлений, ведения дискуссий и конструктивной обратной связи;
Развитие у участников навыков осознанного подхода к собственному поведению и манерам.

## Назначение проекта.

Проект призван решить проблему, которая заключается в разрозненности информации одного проекта - Финатлон. Имеются три разных сайта, которые принадлежат данному проекту, однако куда удобнее было бы сделать единую веб-платформу со всеми функциями, содержащимися отдельно на каждом из сайтов: fin-olimp.ru, fin-victory.ru, finatlonforum.ru. При этом требуется решить проблемы слабого уровня вовлеченности аудитории.
Проект "Финатлон. Веб-платформа для финансовых олимпиад" решает несколько ключевых проблем:
Разрозненность информации — олимпиады представлены на отдельных сайтах, что затрудняет навигацию и поиск актуальных событий для участников.
Отсутствие единого цифрового пространства — нет централизованной платформы для подачи заявок, отслеживания статуса участия и хранения документов.
Недостаточная коммуникация с аудиторией — участники не получают своевременных уведомлений о новых олимпиадах, дедлайнах и изменениях.
Слабый уровень вовлеченности — отсутствие активных сообществ в социальных сетях снижает интерес и лояльность школьников и учителей.
Отсутствие системной работы с партнёрами — нет подготовленной базы контактов и материалов для привлечения спонсоров и информационных партнёров.
Сложность подачи заявок — отсутствие удобной формы с загрузкой файлов создает барьеры для участников и усложняет организаторам сбор данных.
